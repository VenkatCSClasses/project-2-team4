Client Statement:

	The desired product is an e-book reader with a general library functionality. It needs to contain a library of books that can be accessed by users for reading purposes. Users have the ability to search and select books by author, title, or genre from the global library to be added to or removed from their personal library. Additionally, the user should be able to see general stats about their library, such as what books are in it, what books have been started/finished, and a search overview of how much of a given genre/author is in their library. When adding books to their library, a user can view a small preview of a book before adding it. A librarian/admin style user is able to edit, update, and add/remove books from the global library. The librarian/admin also manages user libraries and is able to access user accounts and perform all the same actions. Librarians are also able to have their own “personal account” section, where they are able to access their own personal library akin to a normal user account.
	Books are stored within the library as .epub files, which are parsed and then displayed to the user upon reading. A stretch goal is to provide implementation for standard plaintext files as well.
